🎟️ Event Ticketing API

Use this RESTful API to register, login, browse events, and book tickets. Built with Node.js, Express, and MongoDB.

👤 Auth Endpoints

  • POST /api/auth/register – Register a new user
  • POST /api/auth/login – Login and receive JWT token

📅 Event Endpoints

  • GET /api/events – List all events
  • GET /api/events/:id – Get a specific event by ID
  • GET /api/events?category= – Filter events by category
  • GET /api/events?date=YYYY-MM-DD – Filter events by date
  • POST /api/events – Create new event (admin only)
  • PUT /api/events/:id – Update event (admin only)
  • DELETE /api/events/:id – Delete event and its bookings (admin only)

🧾 Booking Endpoints

  • POST /api/bookings – Book tickets for an event (user only)
  • GET /api/bookings – Get all bookings of logged-in user
  • GET /api/bookings/:id – Get booking by ID (user only)

✨ Optional Features

  • GET /api/bookings/validate/:qr – Validate ticket using QR code
  • GET /api/admin/dashboard – Admin can view who booked each event (admin only)
  • 📧 Sends email confirmation after booking (nodemailer)
